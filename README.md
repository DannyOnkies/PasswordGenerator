# Password Generator
A password generator with Tkinter
Generate a password according to the following scheme:

**`WWWW-N-S-N-LL-S-L`**

* WWWW four letters
* N    one number
* S    one symbol
* N    one number
* LL   two letters
* S    one symbol
* L    one letter

Example : xcgw7/3hx*G , ayxq0)6hz"Q

**`Note`**
The program is mostly a test to try out the Tkinter library
